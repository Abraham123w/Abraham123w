## Proyectos destacados

### Dashboard predictivo para el mantenimiento de cámaras de aguas lluvias

Aplicación web para apoyar la planificación del mantenimiento preventivo de cámaras de aguas lluvias. Combina **machine learning, registros históricos de limpieza y visualización geoespacial** para estimar próximas fechas de intervención, identificar sectores críticos y priorizar tareas operativas — apoyando una asignación más eficiente de cuadrillas, camiones vactor y otros recursos municipales.

**Tecnologías:** Python · Machine Learning · JavaScript · HTML · CSS · Leaflet.js · Chart.js

**[Ver dashboard interactivo](https://abraham123w.github.io/Dashboard_Predictivo_para_Mantenimiento_de_c-maras-Aguas-Lluvias/)**

### Optimización del proceso de evaluación de proyectos con Lean Six Sigma

Proyecto desarrollado para el **Gobierno Regional de La Araucanía**, orientado a identificar las principales causas de rechazo de iniciativas de inversión pública y detectar oportunidades de mejora en el proceso de evaluación técnica.

Con **Python** automaticé la extracción, limpieza y estructuración de información desde documentos PDF. Luego apliqué **minería de texto** y la metodología **Lean Six Sigma (ciclo DMAIC)** para identificar patrones recurrentes, analizar causas raíz y proponer acciones de mejora. El resultado fue una propuesta progresiva de **estandarización, digitalización y analítica avanzada**, orientada a reducir reprocesos, mejorar la trazabilidad de la información y fortalecer la toma de decisiones basada en datos.

**Tecnologías y metodologías:** Python · Minería de datos y texto · Lean Six Sigma · DMAIC · HTML

**[Ver resultados del análisis por etapas DMAIC](https://abraham123w.github.io/Lean-six-sigma-Gore-Araucan-a-/)**
